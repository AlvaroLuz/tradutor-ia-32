Feito por:

O código em C++ foi desenvolvido e testado em um Linux da distribuição Ubuntu versão 20.04.2 LTS e em um Kali Linux versão 2022.4

O tradutor consta no codigo em c++ escrito em trabalho.cpp

O programa pode ser compilado e rodado no linux utilizando os comandos

	g++ programa.cpp -o programa
	./programa <arquivo a ser traduzido>.asm
